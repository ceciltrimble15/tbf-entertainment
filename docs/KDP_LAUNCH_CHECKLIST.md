# KDP Launch Checklist — _Young G's vs. Old G's: The Prefix_

Operational checklist to publish on Amazon KDP and run the 30-day launch. This is the doer's version of the campaign shown on the Publishing page of the site.

---

## Phase 0 — Prep (before KDP)

- [ ] Final manuscript proofread (no placeholder text).
- [ ] Interior formatted to 6×9 trim — PDF (print) + reflowable `.epub`/`.docx` (Kindle). Tools: Kindle Create or Vellum.
- [ ] Cover finalized: front cover JPG (eBook) **and** full wrap PDF (print) sized to KDP's spine calculator for your page count.
- [ ] ISBN decision made and recorded (`ISBN_TRACKING.md`).
- [ ] Author/publisher accounts ready: KDP account, bank + tax (W-9) info entered.
- [ ] Back-cover blurb + book description written (see metadata below).

## Phase 1 — Build (Days 1–7): get it live

### Kindle eBook
- [ ] KDP → **Create** → Kindle eBook.
- [ ] Title: `Young G's vs. Old G's: The Prefix`; Series: `Young G's vs. Old G's` (Book 1).
- [ ] Author: `O.G. Tom Tom`; Publisher: `TBF Entertainment Publishing`.
- [ ] Description (HTML allowed — use the site's hook copy).
- [ ] **Keywords (7 slots):** `urban fiction`, `street lit`, `Cincinnati`, `gang fiction`, `Black fiction`, `crime drama`, `hood novel`.
- [ ] **Categories (3):** Fiction › Urban & Street Lit; Fiction › African American › Crime; Fiction › Crime.
- [ ] Upload manuscript + cover; preview in the online previewer.
- [ ] Price: **$4.99–$6.99**. Decide on **KDP Select / Kindle Unlimited** (90-day exclusivity) by Day 5.
- [ ] Publish.

### Paperback
- [ ] KDP → same title → **Paperback**.
- [ ] ISBN: use owned Bowker ISBN (or KDP-free if Amazon-only).
- [ ] Upload interior PDF + cover wrap PDF; order or download the **digital proof**.
- [ ] Price: **$14.99–$16.99** (check the royalty preview is positive).
- [ ] Publish.

### IngramSpark (expanded print — optional but recommended)
- [ ] Requires an **owned ISBN** (KDP-free won't work here).
- [ ] Upload same interior/cover; enable wholesale discount + returns for bookstore access.

## Phase 2 — Ignite (Days 8–14)
- [ ] Confirm live links; **update the website**: replace the Amazon search URLs in `src/App.jsx` with the real product URL / ASIN, then rebuild + redeploy.
- [ ] Record **ASIN** in `ISBN_TRACKING.md` and Airtable `Titles`.
- [ ] Recruit 10–20 ARC readers (use the site's "Request ARC Copy" → Leads table).
- [ ] Excerpt + cover reveal across social (TikTok/BookTok focus).
- [ ] Set up Goodreads author + book page; optional giveaway.

## Phase 3 — Strike (Days 15–21)
- [ ] Launch-day push: all channels same day.
- [ ] ARC readers post reviews (target **50+** Amazon, **30+** Goodreads).
- [ ] Author live/Q&A; daily BookTok content.
- [ ] Cincinnati bookstore consignment deliveries (Airtable `Retail Outreach`).

## Phase 4 — Hold (Days 22–30)
- [ ] Reader spotlight / UGC reshares.
- [ ] Tease Book 2.
- [ ] Review KDP dashboard: units, KU page reads, ad performance.
- [ ] Build Month-2 plan from the data.
